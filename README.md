A customizable splash screen designer for Adobe After Effects. Create personalized splash screens with custom colors, logos, and artwork.

## Features

- Customizable colors, gradients, and backgrounds
- Upload your own logos and artwork
- Real-time preview with 3D rotation effect
- Export as PNG for use with After Effects
- Step-by-step installation guide

## Run Locally

**Prerequisites:** Node.js

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ae-splash-screen-customizer.git
   cd ae-splash-screen-customizer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to http://localhost:3000

## Build for Production

To build the application for production:

```bash
npm run build
```

## Deployment to GitHub Pages

1. Update the `homepage` field in `package.json` with your GitHub Pages URL
2. Run the deployment command:
   ```bash
   npm run deploy
   ```

## Usage

1. Customize your splash screen using the controls panel
2. Click "Export Design" to download your PNG
3. Follow the installation instructions in the app to replace the splash screen in After Effects
